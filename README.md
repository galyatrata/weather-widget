# 🌤️ Weather Widget — Твій персональний прогноз погоди

Красивий погодний віджет на React з динамічними фонами
та 5-денним прогнозом погоди.

## Стек технологій
- React 18 + Vite
- OpenWeatherMap API
- CSS-in-JS

## Запуск проєкту

### 1. Встанови залежності
npm install

### 2. Додай API ключ
У файлі src/App.jsx знайди рядок:
const API_KEY = "YOUR_API_KEY_HERE";
Заміни на свій ключ з openweathermap.org

### 3. Запусти
npm run dev

Відкрий http://localhost:5173

## Функціонал
- Пошук погоди по місту
- Динамічний фон залежно від погоди
- Температура, вологість, вітер, тиск
- 5-денний прогноз
- Перемикач °C / °F
